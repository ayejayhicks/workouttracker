# Workout Tracker

## Purpose
Build a CMS-style blog site similar to a Wordpress site, where developers can publish blog posts and comment on other developersâ€™ posts as well.

## Description
The Tech Blog app will follow the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.

## Technologies Used
Node.js  
Express.js  
MySQL  
Sequelize  
Heroku/JawsDB
